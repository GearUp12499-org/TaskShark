# TaskShark

## Usage

Add the TaskShark repo to your Gradle configuration:
```groovy
repositories {
    maven {
        url = 'https://gearup12499-org.github.io/TaskShark/'
    }
}
```

And depend on TaskShark:
```groovy
dependencies {
    implementation 'io.github.gearup12499.taskshark:taskshark-android:0.2-SNAPSHOT'
}
```
